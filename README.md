# vue-hexagon-menu

## Usage

![img](https://i.imgur.com/o4onSyp.png?1)

```html
<hex-menu
  :maxLength="3"
  :items="items"
  :classes="['home-styles']"
  color="#6c6"
  hoverColor="#69c"
></hex-menu>
```

![img](https://i.imgur.com/XGK4ACj.png?1)

```html
<hex-menu
  :items="items"
  rotated
  color="#6c6"
  hoverColor="#69c"
></hex-menu>
```
